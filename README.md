# Blood Bank Management System  
*C# Windows Forms Desktop Application*

A desktop-based **Blood Bank Management System** built using **C# and Windows Forms**.  
This project focuses on managing donors, recipients, and blood stock through a simple and structured graphical interface.

It was developed to understand real-world CRUD operations, form-based workflows, and basic system design using .NET.
